# Статические HTML-макеты UI

Макеты соответствуют страницам `frontend/src/pages/*.tsx` и навигации `frontend/src/App.tsx` (тёмная боковая панель слева). Для импорта в Figma откройте нужный `.html` в браузере (файл и `shared.css` должны лежать в одной папке).

## Файлы

| Раздел | Список | Создание (модалка) | Редактирование (модалка) | Прочее |
|--------|--------|--------------------|---------------------------|--------|
| Сотрудники | `employees-list.html` | `employees-create.html` | `employees-edit.html` | |
| Клиенты | `clients-list.html` | `clients-create.html` | `clients-edit.html` | |
| Темы | `themes-list.html` | `themes-create.html` | `themes-edit.html` | |
| Подтемы | `subthemes-list.html` | `subthemes-create.html` | `subthemes-edit.html` | |
| Обращения | `appeals-list.html` | `appeals-create.html` | `appeals-edit.html` | `appeal-detail.html` (страница деталей) |
| Слоты | `slots-list.html` | `slots-create.html` | `slots-edit.html` | |
| Команды | `teams-list.html` | `teams-create.html` | `teams-edit.html` | |
| Воркфлоу | `workflows-list.html` | `workflows-create.html` | `workflows-edit.html` | |

Общие стили: `shared.css`.
